# Agent Avatar Presets

These preset avatars are bundled for the `Agent 团队管理` page only.

License:
- CC0

Human source:
- OpenGameArt / 32 x 32 Portraits
- https://opengameart.org/content/32-x-32-portraits

Notes:
- Presets 01-12 are modern human avatars rebuilt from the CC0 portrait source above.
- Presets 13-20 are original common-animal avatars drawn for this project to match the same pixel style and are released here as CC0.
- The V2 set intentionally uses one unified pixel language:
  modern people on top, common animals on bottom, no repeated animal species.
- `preview-20.png` is only a visual index for quick review.
